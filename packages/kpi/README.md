# My Fancy Button Example

This is an component example.

## Installation

```sh
yarn add "@dex/my-fancy-button
```

## Usage

Global (scoped) css must be imported once (in the root `index.tsx` of your where `@sg-bootstrap/icons/dist/index.css` is also imported)

```typescript
import '@dex/my-fancy-button/dist/index.css';
```
